# OPCode 13 - Call sync
OPCode 13, sometimes known as "Call sync", is called by the client to get
the voice state of users in a given DM/group channel.

You will have to send a [Call Create](events/CallCreate) event if there is
one or more voice states for the channel.

## Call sync request body
| field | type | description |
| --: | :-- | :-- |
| channel_id | string | the channel the client wants the status of |

## Misc
This is called "CALL_CONNECT" by the official client
